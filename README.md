#  AI Traffic Congestion Predictor

##  Overview
This project is an AI-based system that predicts traffic congestion levels using machine learning techniques. It analyzes factors like vehicle count, speed, weather, and peak hours to provide accurate predictions.

---

##  Features
- Data Analysis & Visualization (EDA)
- Machine Learning Model (Random Forest)
- Interactive Streamlit Web App
- Real-world traffic simulation dataset

---

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Streamlit

---

##  How to Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/AI-Traffic-Predictor.git
cd AI-Traffic-Predictor
pip install -r requirements.txt
streamlit run app/app.py
